HiResCAM

HiRes-CAM (High-Resolution Class Activation Mapping) is a technique used to generate Class Activation Maps (CAM) for a convolutional neural network (CNN). It is an improvement over the original CAM and Grad-CAM methods, which only use the feature maps of the last convolutional layer to generate the heat map.
HiRes-CAM method aims to overcome the limitations of the previous CAM techniques by using the feature maps from all the layers in the CNN to generate a more detailed and high-resolution heat map.
The main idea behind HiRes-CAM is to combine the feature maps of all the layers in the CNN to obtain a more detailed and high-resolution heat map. To achieve this, the method first generates CAM heat maps for each layer in the CNN, and then uses an upsampling process to increase the resolution of the heat maps. Finally, the heat maps are combined to obtain a final high-resolution heat map.
HiRes-CAM is particularly useful for understanding the decision-making process of a CNN because it allows us to see which regions of the image are important to the network's prediction, even when the final decision is made by layers deep in the network.
the HiResCAM explanation for a model with a CAM architecture is:

A m H i R e s C A M = 1 D 1 D 2 ∑ f = 1 F w m f A f

the gradient ∂ S m ∂ A ( f ) :

∂ S m ∂ A ( f ) = 1 D 1 D 2 w m f
How do I use this technique on an image?
To load a HiResCAM class:

This technique have Two parameters:

  • model: This parameter refers to the convolutional neural network (CNN) that is being used to make predictions.
  • target_layer: This parameter refers to the specific layer of the CNN that the gradients are computed with respect to.(Optional)
from tensorcam import HiResCAM
from tensorflow.keras.applications import ResNet50
import numpy as np
model = ResNet50(weights='imagenet')
target_layer= 'conv5_block3_out'

This technique have Three methods :

get_image: function takes the origenal image and retern the image as a numpy 2d array and the origenal image self.
compute_cam_features: function takes an input image,and returns the CAM heat map for that class.
plot_images: that takes the origenal image , the heatmap and return the image with the heatmap.

image, orig = HiResCAM.get_image('sample.jpg')
preds = model.predict(image)
class_idx = np.argmax(preds[0])
hirescam = HiResCAM(model, class_idx,target_layer)
heatmap = hirescam.compute_cam_features(image)
hirescam.plot_images(orig,heatmap)
