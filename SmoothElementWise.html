SmoothElementWise

We use the Smooth technique to improve the quality of the GradCAMElementWise method

How do I use this technique on an image?
To load a Smoothgradelementwise class:

This technique have Four parameters:

  • model: This parameter refers to the convolutional neural network (CNN) that is being used to make predictions.
  • target_layer: This parameter refers to the specific layer of the CNN that the gradients are computed with respect to.(Optional)
  • N_SAMPLES: This parameter controls the number of noisy versions of the input image that are generated.
  • STD_DEV: This parameter controls the standard deviation of the noise that is added to the input image.
from tensorcam.SmoothElementWise import Smoothgradelementwise
from tensorflow.keras.applications import ResNet50
import numpy as np
model = ResNet50(weights='imagenet', include_top=True)
target_layer= 'conv5_block3_out'
N_SAMPLES = 10
STD_DEV = 0.15
This technique have Two methods :

get_image:that takes the origenal image and retern the image as a numpy 2d array and the origenal image self.
plot_images:that takes the origenal image,the heatmap and return the image with the heatmap.

image,orig=Smoothgradelementwise.get_image('sample.jpg')
X = preprocess_input(image)
prediction = np.argmax(model.predict(X))
CAM = Smoothgradelementwise(model ,prediction, target_layer,N_SAMPLES,STD_DEV)(image)
Smoothgradelementwise.plot_images(orig,CAM)
