Class Activation Maps (CAM)

Class Activation Maps (CAM) is a technique used to visualize the regions in an image that are most important for a convolutional neural network (CNN) to classify a specific class.
CAM generates heat maps that highlight the regions in an image that contribute the most to the final prediction made by the CNN.
These heat maps are generated by taking the dot product of the feature maps of the last convolutional layer of the CNN with the weight vector of the output class.
This highlights the regions in the image that are most important for the CNN to make its prediction for that class.
CAM is particularly useful for understanding the decision-making process of a CNN and for identifying the regions in an image that are most relevant for a particular class.
The localization map is computed as follows:

L C A M ( c ) ( x , y ) = R e L U ( ∑ k w k ( c ) A k ( x , y ) )

where A k ( x , y ) is the activation of node in the target layer of the model at position ( x , y ) , and w k ( c ) is the weight corresponding to class for unit in the fully connected layer.

How do I use this technique on an image?
To load a BaseCAM class:

This technique have Two parameters:

  • model: This parameter refers to the convolutional neural network (CNN) that is being used to make predictions.
  • target_layer: This parameter refers to the specific layer of the CNN that the gradients are computed with respect to.(Optional)
from tensorcam import BaseCAM
from tensorflow.keras.applications import ResNet50
model = ResNet50(weights='imagenet')
target_layer= 'conv5_block3_out'
cam = BaseCAM(model,target_layer)

This technique have 3 methods :

get_image: function takes the origenal image and retern the image as a numpy 2d array and the origenal image self.
compute_cam_features: function takes an input image,and returns the CAM heat map for that class.
plot_images: that takes the origenal image , the heatmap and return the image with the heatmap.

image, orig = cam.get_image('sample.jpg')
cam_features = cam.compute_cam_features(image)
cam.plot_images(orig, cam_features)
